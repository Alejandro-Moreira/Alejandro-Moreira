# 👋 Sobre mí

Hola, soy **Alejandro Moreira**
Desarrollador de software enfocado en construir aplicaciones web modernas, escalables y orientadas a la experiencia de usuario.

Trabajo en soluciones de **E-commerce **, integrando:

* Arquitecturas SPA
* Sistemas de autenticación robustos
* Dashboards con analítica en tiempo real

# Proyectos Destacados

## Plataformas E-Commerce

* SPA con React + Vite
* Backend con Node.js + Express
* Autenticación JWT + seguridad avanzada
* Dashboard administrativo + analítica
* Sistema de recomendaciones y tracking de usuarios

---

# Contribuciones Destacadas

*  Implementación de **autenticación segura con JWT + bcrypt**, incluyendo verificación de correo y recuperación de contraseña.
*  Desarrollo de **dashboard administrativo con métricas en tiempo real** (ventas, pedidos, rendimiento).
*  Diseño de **sistema de recomendaciones de productos** basado en comportamiento del usuario.
*  Integración de **analítica de interacción (heatmaps)** para mejorar UX.
*  Modelado de base de datos relacional optimizada para **gestión de productos, variantes e inventario**.
*  Construcción de SPA fluida con **React + Vite**, optimizando tiempos de carga y navegación.
